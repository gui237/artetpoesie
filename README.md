# Art & Poésie

Site web artistique et poétique créé avec HTML, CSS et JavaScript.

## Technologies
- HTML5
- CSS3
- JavaScript
- Netlify
- Supabase

## Site en ligne
https://artetpoesie.com

## Objectif
Créer une plateforme poétique interactive où les visiteurs peuvent proposer des poèmes.
